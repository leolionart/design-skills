# Doctrine

## Thesis
Break safe defaults on purpose. Use tension, contrast, and authored disruption while keeping one hierarchy spine intact.

## Visual grammar
- one dominant disruptive move
- visible tension in type, borders, or alignment
- contrast as authorship, not chaos
- structure under pressure
- memorability over neutrality

## Typography
- type can be oversized, raw, glossy, or hard-edged
- hierarchy must still remain readable
- expressive choices need one anchor of order

## Components
- buttons, cards, and nav can break politeness
- repeated defaults should be challenged
- statement sections should feel authored, not randomized

## Motion
Use restrained-to-disruptive motion depending on the concept. Motion should heighten tension, not muddy control.

## Anti-patterns
- every element shouting equally
- random misalignment
- loud palette with weak hierarchy
- anti-design with no thesis
- novelty that destroys task flow
