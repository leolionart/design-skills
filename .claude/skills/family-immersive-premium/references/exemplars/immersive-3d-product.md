# Immersive 3D Product

## Thesis
Build the page around one staged object or rendered product scene so the interface feels like a product reveal rather than a standard landing page with effects.

## Best when
- the brief says 3D product, object-centric, rendered scene, reveal, hardware, or wow-factor launch
- the hero should revolve around one product object
- premium spectacle is needed from the first screen
- motion tolerance is medium

## Distinguishing signals
- more object-centered than `dark-glow-aurora`
- less system-surface focused than `glassmorphism-mature`
- less pacing-led than `motion-led-storytelling`
- less material-sheen focused than `iridescent-holographic-chrome`

## Layout directives
- center the page around one primary object or rendered scene
- let supporting sections feel like reveal chapters around that object
- reduce competing focal elements so the object stays dominant
- use large framing and depth cues to support the product stage

## Typography and surface directives
- keep typography subordinate to the staged object
- use surfaces, light, and contrast to frame the object scene
- let background atmosphere support the reveal without stealing attention
- use premium accents sparingly around the focal object

## Component behavior
- hero carries most of the spectacle load
- CTA should sit within the object narrative, not outside it
- feature sections can feel like reveal frames or close-ups
- cards and proof panels should stay quieter than the hero scene

## Avoid
- multiple competing hero objects
- using 3D only as decoration with no structural role
- narrative pacing so strong that the object stops being the center
- burying CTA and proof under visual wow-factor

## Review and conversion cues
- If the premium feel comes mostly from dark glow and atmosphere, it is drifting toward `dark-glow-aurora`.
- If the page is really structured as a reveal journey rather than an object stage, it is drifting toward `motion-led-storytelling`.
- To convert from flatter launches, choose the hero object first before adding more depth effects.
