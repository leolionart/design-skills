# Doctrine

## Thesis
Let type, spacing, and restraint carry the visual identity. Make the page feel composed, expensive, and reading-aware.

## Visual grammar
- wide breathing room
- calm section cadence
- low decorative noise
- typography before chrome
- refined contrast, not loud contrast

## Typography
- headlines carry brand tone
- serif-sans pairing is often useful
- scale should feel deliberate, not busy
- body copy needs readable rhythm

## Components
- CTA stays clear but calm
- cards, borders, and pills stay restrained
- supporting UI should frame text, not dominate it

## Motion
Keep motion static-to-restrained. Use motion to support reading flow, never to overwhelm it.

## Anti-patterns
- gradient-heavy decoration
- pill spam
- too many boxed sections
- loud accent colors with quiet type
- luxury cues without spacing discipline
