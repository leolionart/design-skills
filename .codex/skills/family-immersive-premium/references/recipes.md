# Recipes

## Exemplar chooser
- `glassmorphism-mature` -> use for layered translucent UI with premium system-surface depth.
- `dark-glow-aurora` -> use for cinematic dark launches with luminous atmosphere.
- `immersive-3d-product` -> use when a product object or hero object should become the center of gravity.
- `motion-led-storytelling` -> use when narrative pacing and reveal structure matter more than a single material treatment.
- `iridescent-holographic-chrome` -> use when reflective materiality, chrome sheen, or fashion-tech novelty define the mood more than darkness or glass.

## Quick differentiation rules
- Choose `iridescent-holographic-chrome` over `dark-glow-aurora` when the brief is reflective, fashion-tech, beauty-tech, or music-led rather than dark cinematic.
- Choose `glassmorphism-mature` over `dark-glow-aurora` when the UI still needs strong system readability on light or mixed backgrounds.
- Choose `motion-led-storytelling` over `immersive-3d-product` when pacing and reveal matter more than a single object hero.
- Choose `immersive-3d-product` only when a product object truly deserves center stage.

## Safe blends
- `glassmorphism-mature` + grid-product for readable premium product pages
- `dark-glow-aurora` + high-agency for flagship tech launches
- `iridescent-holographic-chrome` + experimental-loud for campaign-like fashion-tech energy

## Escalation rule
Select the immersive mode by asking what creates the premium feel: translucent surfaces, dark atmosphere, object staging, narrative pacing, or reflective materials. Open the chosen exemplar pack only after that source of spectacle is clear.
