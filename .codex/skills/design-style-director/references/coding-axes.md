# Coding Axes

Translate style words into implementation decisions through these axes.

## 1. Hierarchy
Decide whether the page should feel commanding, quiet, modular, cinematic, soft, or disruptive.
- commanding -> high-agency
- quiet -> editorial-typography
- modular -> grid-product
- cinematic -> immersive-premium
- soft -> tactile-organic
- disruptive -> experimental-loud

## 2. Layout rhythm
State section rhythm clearly.
- high-agency: varied cadence, asymmetry, sharp priority shifts
- editorial-typography: long breathing room, reading cadence, restrained shifts
- grid-product: modular scan pattern, grouped proof, dense-but-clear blocks
- immersive-premium: reveal staging, focal scenes, layered transitions
- tactile-organic: rounded grouping, gentle flow, low-friction progression
- experimental-loud: controlled clashes, tension, deliberate interruptions

## 3. Typography
Specify dominance and voice.
- editorial families need typography to carry identity
- product families need typography to rank information fast
- immersive families need typography to support atmosphere without losing readability
- loud families need typography to create tension without destroying order

## 4. Density and spacing
Always state one target:
- airy
- balanced
- dense-but-readable
- dramatic contrast

## 5. Surfaces and borders
Choose a material model.
- flat
- paper-like
- modular cards
- glass/luminous layers
- soft tactile depth
- hard-edge poster logic

## 6. Motion
Set motion budget early.
- static
- restrained
- supportive
- narrative
- playful
- disruptive

## 7. Components
Describe treatment for hero, nav, cards, CTAs, proof blocks, and forms.
Do not stop at palette language.

## 8. Anti-patterns
Always include what to remove:
- generic centered hero
- repeated equal cards
- decorative glow with no hierarchy
- premium look without restraint
- loudness without structure
- softness without contrast

## 9. Response style
When giving coding directives:
1. state the family thesis
2. state section-level decisions
3. state component-level treatment
4. state motion/material choices
5. state avoid list
